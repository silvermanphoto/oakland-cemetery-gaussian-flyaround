# Oakland Cemetery — 36 full-resolution tiles, live on SuperSplat

All 36 native-8192 gaussian-splat tiles published to Joel Silverman's SuperSplat
account (**silvermanphoto**) as **unlisted** scenes on 2026-07-20. Each tile was
centroid-centered at the origin (splat-transform `-t xc,-9,-zc -s 0.05`, raw trainer
orientation kept) so it opens framed on its own geometry, then injected from R2 into
the SuperSplat editor and published as a New Scene. These replace the earlier Oakland
scenes that were deleted.

Naming rule: a file `tile_A_B` occupies **grid cell (x = 5 − A, z = B)** on the 6×6 grid.
